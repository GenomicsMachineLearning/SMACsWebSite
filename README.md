# SMACs - Cell-Cell Interactions in Aging Web Site

## Running Locally

Steps to run locally include:
* Install Node and Yarn.
* Install JavaScript/TypeScript dependencies.
* Run server and open browser.

1. Clone this repository
```bash
https://github.com/GenomicsMachineLearning/SMACsWebSite.git
```
2. Install dependencies
```bash
$ yarn install
```

3. Start the project

Locally:
```bash
$ yarn build
$ yarn dev
```

Production
```bash
$ yarn build
$ yarn run 
```
